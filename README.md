# Aswin portfolio — Jim Fan format + JobOwl visual style

This version uses:
- a structure inspired by Jim Fan's site:
  - Home
  - Featured
  - Media
  - Publications
  - Experience
  - Contact
- a visual direction inspired by JobOwl:
  - light gray background
  - Inter font
  - neon green highlight
  - vivid blue CTA buttons
  - bold black outlines and offset shadows

## Deploy
1. Create a public GitHub repo named `yourusername.github.io`
2. Upload `index.html` and `style.css`
3. Open **Settings → Pages**
4. Choose:
   - Deploy from branch
   - Branch: `main`
   - Folder: `/ (root)`

Your site will be live at:
`https://yourusername.github.io`
